# Автоматизация тестирования сайта "Читай-Город"

## Описание проекта
Автоматизированное тестирование UI и API сайта интернет-магазина книг "Читай-Город" (chitai-gorod.ru).

Проект создан на основе финальной работы по ручному тестированию и включает:
- 5+ UI тестов (smoke и функциональные)
- 5+ API тестов (на основе Postman коллекции)
- Тестирование производительности (время загрузки страниц)
- Кроссбраузерное тестирование

## Структура проекта

chitai-gorod-automation/
├── config/ # Конфигурационные файлы
├── data/ # Тестовые данные и учетные данные

## Установка и настройка

1. Клонировать репозиторий:
```bash
git clone <repository-url>
cd chitai-gorod-automation

2. Создать виртуальное окружение:
python -m venv venv
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

3. Установить зависимости:
pip install -r requirements.txt

4. Настроить переменные окружения:
cp .env.example .env
# Отредактировать .env файл, указав необходимые данные

## Запуск тестов

# Запуск всех тестов
pytest
# Запуск только UI тестов
pytest -m ui
# Запуск только API тестов
pytest -m api
# Запуск smoke тестов
pytest -m smoke
# Запуск с генерацией Allure отчетов
pytest --alluredir=allure-results
allure serve allure-results
# Запуск Allure отчёта из директории skypro_automation:
.\allure-2.36.0\bin\allure.bat serve .\chitai-gorod-automation\allure-results
# Запуск после добавления Allure в переменную PATH:
allure serve
# Параллельный запуск тестов
pytest -n 4
